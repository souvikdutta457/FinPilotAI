# FinPilot AI

## Vision

FinPilot AI is an AI-powered personal finance platform designed to simplify budgeting and financial analysis.

Instead of manually entering expenses into spreadsheets, users can interact with the system using natural language.

Example:

Spent ₹450 on Domino's.

The system understands the transaction, categorizes it automatically, updates the Excel workbook, refreshes the dashboard and provides intelligent financial insights.

The goal of this project is to demonstrate how Artificial Intelligence can improve personal finance management while showcasing practical applications of Microsoft Excel, automation and financial analytics.

---

## Why I Built This

As a BCom (Finance & Investment) student, I wanted to build something that combines finance with modern technology.

Rather than creating another expense tracker, I wanted to develop a system that behaves like a real financial assistant capable of understanding transactions, analysing spending patterns and generating meaningful insights.

This project also serves as a practical demonstration of AI integration, automation and business intelligence.
